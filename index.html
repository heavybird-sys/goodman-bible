About

Divine Words (Scaled) is an offline-first, private, and customizable Bible study and presentation tool.

This application was developed by an advanced AI model to create a powerful, framework-free web experience.

Help

  • Import Data: Use the 'Tools' tab to open the Data Manager. From there you can import Bible XML files and create verse collections.
  • Start Presenting: Select a Bible and a Collection from the main menu (⋮). Use the arrow keys or on-screen controls to navigate.
  • Customize: Use the 'Personalize' tab to change themes, fonts, and audiovisual settings.
